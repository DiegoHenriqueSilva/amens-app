# UI/UX Rules - Améns

## 🎨 Design System Enforcement

- ALWAYS check `DESIGN-SYSTEM.md` before:

  - Changing layout
  - Modifying styles

---

- Any new UI element MUST be discussed in:
  @brainstorming-uiux.md

- New pages
- Buttons
- Layout changes

## 🧩 Implementation

- All UI implementations must follow:
  @agent-frontend-design.md

## 📱 UX Principles

- Keep UI simple and intuitive
- Prioritize emotional clarity (important for religious context)
- Avoid clutter
- Use clear call-to-actions

## ✝️ Tone & Content

- Always respectful and empathetic
- Avoid aggressive or commercial tone
- Maintain spiritual alignment
